The home mango is first registered when mango is installed, and is located at `~/.mango`.

Default setup of home mango:

~/.mango
├── .instructions
├── .submodules
│   └── builtins
|       ├── .mango
|       |   ├── .templates.registry
|       |   |   └── (template folders)
│       |   ├── .instructions
│       |   ├── .on-add
│       |   ├── .on-add.literal
|       |   ├── .on-install
│       |   └── (builtin mango commands)
|       └── (license and readme files)
├── .templates.registry
|   └── (user defined template folders)
├── .submodules.registry
|   └── (user defined submodules)
└── (user defined mango scripts)

The creation of the home mango is handled by the installation script, where the mango submodule is cloned into the `.submodules/builtins` folder, and the .on-install script is executed.
